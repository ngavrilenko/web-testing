# Тестирование веб-приложений

В рамках тестирования веб-приложения [«Интернет-магазин»](https://demoshopping.ru/) разработала полный комплект тестовой документации: тест-план, тест-кейсы, чек-листы и отчёты о дефектах. Тест-план определял стратегию и приоритеты тестирования, а тест-кейсы — проверку функциональности на основе техник тест-дизайна. Чек-листы использовались для быстрой регрессии, а дефекты фиксировались в системе YouTrack. Это позволило наладить процесс тестирования и улучшить качество продукта.
